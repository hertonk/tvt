
## Trace View Tool 

O Trace View Tool (TVT) é um protótipo de visualização de rastramento de software desenvolvido em uma pesquisa de Mestrado do Programa de Pós-Graduação em Informática Aplicada (PPGIA) da Universidade Federal Rural de Pernambuco (UFRPE) pelo aluno Herton Freire Vilarim, sob a orientação do Professor GILBERTO AMADO DE AZEVEDO CYSNEIROS FILHO.
